# AGENTS.md

## Intent

This repository is a clean Flutter baseline for rebuilding MemoX from scratch.

## Working rules

- Keep changes minimal and explicit.
- Do not assume the old MemoX architecture, docs, or guard system still apply.
- Introduce new layers only when the repo actually contains them.
- Prefer simple Flutter structure until a new architecture is defined.

## Verification

After code changes, run:

```bash
flutter analyze
flutter test
```
